# Teslo API

1. Clonar Proyecto
2. ```yarn install```
3. cambiar las variables de entorno ```
.env.template```
4. Levantar la base de datos 

```
docker-compose up -d
```

6. Levantar proyecto ``` yarn start:dev```

7. Ehjecurtar Seed

```
localhost:3000/api/seed
```
